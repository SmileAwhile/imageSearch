ImageSearch

Instructions:

Enter search term in /imgsearch/ path and get image search results
Add ?offset=[number] to paginate through search results
Or navigate to /latest/ to see latest search terms

Example search:

https://imsearc.herokuapp.com/imgsearch/steve?offset=2

Example search output:

[{"imgurl":"http://www.ips-dc.org/wp-content/uploads/2014/04/Steve-Cobble.jpg","snippet":"Steve Cobble","pageurl":"http://www.ips-dc.org/authors/steve-cobble/"}, {...}]

latest example:

https://imsearc.herokuapp.com/latest/

latest result example:
